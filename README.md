# Personal Information Manager

A simple Python program to store and display personal information, including name, age, city, and hobbies. This project demonstrates basic input/output handling, string formatting, and working with lists.

## Features

### Version 1: Basic Input and Display
- Takes user input for name, age, city, and hobbies.
- Displays the information in a simple format.

### Version 2: Formatted Output with Multiple Hobbies
- Accepts multiple hobbies in a single line (comma-separated).
- Formats the output neatly.
- Splits hobbies into a list and displays each hobby individually.

## Usage

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/personal-information-manager.git
```

2. **Navigate to the project folder:**
   
```bash
cd personal-information-manager
```


3. **Run the Python script:**
```bash
python personal_info_manager.py
```


Follow the prompts to enter your personal details.

**Example Output**

**Version 1:**
```text
Enter your name: Siddhi
Enter your age: 20
Enter your city: Pune
Enter your hobbies: Reading, Painting

--- Personal Information ---
Name   : Siddhi
```

**Version 2:**
